### Automatic Deployment (Recommended)

The website is automatically deployed to GitHub Pages when you push changes to the repository.

### Manual Deployment

1. Ensure the `docs/` folder is in your repository
2. Go to your repository settings on GitHub
3. Scroll to "GitHub Pages" section
4. Under "Source", select "Deploy from a branch"
5. Choose `main` branch and `/docs` folder
6. Click Save

Your website will be available at: `https://HongSengheng.github.io/Movie4K/`
